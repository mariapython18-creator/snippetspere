# Snippets Sphere

## Overview
**Snippets Sphere** is a coding, sharing, and collaborative platform built with **Python and Django**. It allows students, developers, and experts to share code snippets, ask questions, and collaborate on projects. The platform includes a discussion room feature, code review system, Python compiler, and direct connection options for mentorship or collaboration.
## Features

### **User Features**
- Share and browse code snippets
- Ask questions and get answers from experts or developers
- Apply code snippets to their own projects
- Participate in discussion rooms for collaborative projects
- Run Python code using an integrated online compiler
- Connect with developers and experts for doubts or collaboration
- Review code snippets shared by others

### **Expert & Developer Features**
- Answer user questions and provide guidance
- Participate in collaborative discussion rooms
- Review and suggest improvements for submitted code
- Mentor students and new developers via platform connections

### **Admin Features**
- Manage users, snippets, and discussion rooms
- Moderate content and reviews
- Dashboard with statistics on active users and submissions

### **Common Features**
- Full CRUD operations for snippets, discussions, and user profiles
- Real-time collaboration support
- Responsive UI for web and mobile devices
## Technical Features
- **Asynchronous Discussion:** Discussion rooms use **JavaScript Fetch API** to fetch and submit comments, code suggestions, and snippets in real-time.
- Real-time collaboration for multiple participants without refreshing the page.
- Full CRUD support for discussions, code snippets, and user profiles.

## Technology Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** SQLite / MySQL
- **Code Compiler:** Python online compiler
- **Version Control:** Git & GitHub
